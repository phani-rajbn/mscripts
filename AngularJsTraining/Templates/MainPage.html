Angular Forms

AngularJS uses the features of HTML5 to provide dynamic features that could be used in forms. This include fetures like submitting, validating and redirecting features. AngularjS forms are made with a set of events that could be handled in the code to make it more interactive before the data is submitted to the server or any database resource. Validation of the controls are done using any of the followings states before the form is submitted to the Application

  • $pristine used to check whether the user is yet to interact with the control
  • $dirty used to check whether the user has interacted with the controls
  • $valid checks the validity of all the controls in the form and evaluates whether the form is valid or not
  • $invalid is opposite and will be true even atleast one containing form element is invalid
